# Excel-Practice-Assignments
## 📌 Description
This repository contains a collection of Excel practice assignments designed to build and strengthen fundamental Excel skills. It includes various tasks focused on data handling, formulas, and basic analysis.

## 🚀 Topics Covered
- Basic Excel operations  
- Data entry and formatting  
- Use of formulas (SUM, AVERAGE, IF, etc.)  
- Data sorting and filtering  
- Basic data analysis
- Pivot Tables and Slicers

## 🛠️ Tools Used
- Microsoft Excel  

## 🎯 Purpose
The purpose of this repository is to practice and improve core Excel skills by working on structured assignments and applying fundamental concepts.

## 📚 Learning Outcomes
- Understanding of Excel interface and functions  
- Improved data organization skills  
- Ability to perform basic calculations and analysis  
- Hands-on experience with real datasets
